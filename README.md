# Real-Time Behavior Analytics Platform

A full-stack behavior analytics platform that tracks user events and provides real-time and batch analytics through a dashboard.

## Project Structure

- `backend/` - API, event ingestion, analytics, Redis, workers
- `ecommerce-demo/` - demo frontend/store app
- `public/` - static assets if applicable

## Features

- API-key-based event ingestion
- Event tracking and analytics
- Redis caching
- BullMQ background jobs
- MongoDB storage
- Dashboard visualization
- JWT authentication
- Real-time and batch processing

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Redis
- BullMQ
- JavaScript
- HTML/CSS
- Chart.js
